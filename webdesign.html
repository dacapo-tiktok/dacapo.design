Website & Urnen Design

Für das Startup Urna-Rex entwickelte ich eine komplette Website. Alles entstand in reiner Handarbeit mit HTML, CSS und JavaScript. Highlight des Projekts: ein interaktiver Urnen-Generator, der individuelle Gestaltung in Echtzeit ermöglicht. Auch die Urnen-Designs stammen von mir. Ergebnis: eine stilvolle, nutzerfreundliche Lösung mit emotionalem Mehrwert.

Meine Leistungen:

  • Moodboard, Wireframe, Screendesign
  • HTML, CSS & JavaScript Codeerstellung
  • Responsive Webdesign
  • Bloggestaltung mit ChatGPT
  • Interaktiver Urnen Generator mit Claude & Gemini

Website & Onepager

Im Rahmen meines Studiums entstand die Website für die fiktive Webagentur DUPLEX. Ziel war es, die Vielfalt moderner Webtechniken praktisch umzusetzen – von Lightbox und Slider bis hin zu animierten Buttons. Jedes Feature wurde handgecodet, ohne KI oder Vorlagen. Das Resultat: eine klare, detailverliebte Website, die Funktionalität und Ästhetik gleichermaßen in den Mittelpunkt stellt.

Meine Leistungen:

  • Moodboard, Wireframe, Screendesign
  • HTML, CSS & JavaScript Codeerstellung
  • Responsive Webdesign
